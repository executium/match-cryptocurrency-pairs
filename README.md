# Pair Matching Engine: Discover where you can trade a pair
This repository concentrates on the `endpoint` `POST /api/v2/public/match-pair` designed to provide advice and inspiration for where you can trade specific pairs. The method is of particular use if you are planning to arbitrage a specific pair.
